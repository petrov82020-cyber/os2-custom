    🎥 Видео-инструкция: Как создать сайт на GitHub Pages с нуля

    Источник: YouTube-канал "The Common Coder"[reference:3]

    🎬 Первый шаг: посмотрите видео-введение в GitHub Pages (7 минут) – узнайте, как бесплатно хостить свои проекты.

    📚 План курса

    1. Глава 1: Регистрация на GitHub и создание репозитория – пошаговая инструкция для новичков.
    2. Глава 2: Создание HTML-страницы и загрузка на GitHub – пишем первую веб-страницу и публикуем её.
    3. Глава 3: Основы Git и командной строки. Добавление изображений и CSS – работа с большими файлами, картинки, внешние стили.
    4. Глава 4: Пользовательский домен, темы Jekyll и автоматическая сборка – профессиональный домен, блог на Markdown, GitHub Actions.
    5. Глава 5: Итоги, продвижение, формы обратной связи и ресурсы – завершающая глава с практическими советами.
    6. Глава 6: Введение в JavaScript – оживляем сайт – интерактивность, события, калькулятор.
    7. Глава 7: Карты, формы, отправка писем без сервера – и обзор курса – интерактивные карты, обратная связь, повторение всего материала.
