📘 Экспресс-курс: создание сайта с нуля на GitHub Pages

Добро пожаловать! Этот курс поможет вам создать собственный статический сайт бесплатно и без лишних сложностей. Вы научитесь работать с GitHub, писать простой HTML/CSS и публиковать страницы в интернете. Курс рассчитан на 1–2 часа практической работы.

🎯 Цель: к концу курса у вас будет рабочий сайт по адресу https://ваше_имя.github.io/название_репозитория/.

🎥 Видео-инструкция: Как создать сайт на GitHub Pages с нуля

Источник: YouTube-канал "The Common Coder"[reference:3]

📚 План курса

  1. Регистрация на GitHub и создание репозитория – основы работы с платформой.
  2. Включение GitHub Pages – выбор ветки и папки для публикации.
  3. Создание главной страницы (index.html) – базовые теги HTML.
  4. Добавление стилей (CSS) – как сделать страницу красивой.
  5. Загрузка файлов на GitHub (без командной строки) – через веб-интерфейс.
  6. Работа с Git через командную строку (для больших файлов) – когда веб-интерфейса недостаточно.
  7. Создание простого, но красивого сайта – примеры и шаблоны.
  8. Публикация и поддержка – как обновлять сайт и что делать при ошибках.

🚀 Начните прямо сейчас
