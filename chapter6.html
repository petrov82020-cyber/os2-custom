Глава 6: Введение в JavaScript – оживляем сайт

✍️ Время чтения: ~20 минут | 🎯 Цель: научиться добавлять интерактивность: кнопки, события, динамическое изменение содержимого

До сих пор ваш сайт был статичным – как страница в книге. JavaScript позволяет сделать его живым: реагировать на клики, ввод данных, таймеры, анимации. В этой главе мы напишем первые скрипты и внедрим их в ваш сайт.

💡 Что такое JavaScript?
JavaScript – это язык программирования, который работает в браузере. Он может:
– изменять содержимое страницы (текст, картинки, стили) без перезагрузки;
– обрабатывать события (клики, движение мыши, нажатия клавиш);
– отправлять запросы на сервер (получать данные без перезагрузки);
– создавать игры, калькуляторы, чаты и многое другое.

6.1 Подключение JavaScript к странице

Есть два основных способа:

  1. Встроенный скрипт – код пишется внутри тега <script> в HTML-файле.
  2. Внешний файл – код выносится в файл script.js и подключается через <script src="script.js"></script> (рекомендуется для больших проектов).

Пример встроенного скрипта, который выводит приветствие в консоль браузера (нажмите F12 → Console):

<script>
    console.log("Сайт загружен!");
</script>

6.2 Переменные, типы данных и функции

Базовые понятия, без которых не обойтись:

  • Переменная – «коробка» для хранения данных. Объявляется через let или const (константа).
  • Типы данных: числа (42), строки ("Привет"), булевы (true/false), массивы ([1,2,3]), объекты ({name:"Иван"}).
  • Функция – блок кода, который можно выполнить по имени. Объявляется через function имя() { ... }.
// Пример переменной и функции
let userName = "Анна";
function showMessage() {
    alert("Привет, " + userName);
}
showMessage(); // вызовет всплывающее окно

6.3 События: реакция на действия пользователя

Самый частый сценарий – выполнить код при клике на кнопку. Для этого используется атрибут onclick или метод addEventListener.

Простейший пример (вставьте на любую страницу):

<button onclick="alert('Вы кликнули!')">Нажми меня</button>

Этот код работает прямо в этой главе. Нажмите – увидите всплывающее окно.

6.4 Изменение содержимого страницы

JavaScript может находить элементы на странице по id, классу или тегу и менять их содержимое.

Допустим, у вас есть абзац:

<p id="demo">Старый текст</p>
<button onclick="document.getElementById('demo').innerHTML = 'Новый текст!'">Изменить</button>

6.5 Условные операторы и циклы

if – выполняет код, если условие истинно:

let age = 18;
if (age >= 18) {
    alert("Вы совершеннолетний");
} else {
    alert("Вы ещё ребёнок");
}

for – повторяет код несколько раз:

for (let i = 1; i <= 5; i++) {
    console.log("Шаг номер " + i);
}

6.6 Практический пример: простой калькулятор

Создадим два поля ввода и кнопку, которая складывает числа и выводит результат.

<input type="number" id="num1" placeholder="Первое число">
<input type="number" id="num2" placeholder="Второе число">
<button onclick="sum()">Сложить</button>
<p id="result">Результат:</p>

<script>
function sum() {
    let a = parseFloat(document.getElementById('num1').value);
    let b = parseFloat(document.getElementById('num2').value);
    let s = a + b;
    document.getElementById('result').innerHTML = "Результат: " + s;
}
</script>

6.7 Где размещать JavaScript на странице?

Рекомендуется размещать тег <script> перед закрывающим тегом </body>, чтобы браузер сначала отобразил HTML и CSS, а затем загрузил скрипты. Это ускоряет загрузку страницы.

<body>
    ... весь контент ...
    <script src="script.js"></script>
</body>
📝 Задание для самопроверки
1. Добавьте на одну из страниц вашего сайта кнопку, которая при клике выводит текущую дату и время (используйте объект new Date()).
2. Создайте простую форму с полем ввода имени и кнопкой. При нажатии должно выводиться приветствие «Привет, [имя]!».
3. Напишите функцию, которая меняет цвет фона страницы при клике на кнопку (например, с белого на голубой).
4. Поместите код в отдельный файл script.js и подключите его к странице.

6.8 Отладка: как искать ошибки

Никто не пишет код без ошибок. В браузере есть отличный инструмент – консоль разработчика (F12 → вкладка Console). Ошибки JavaScript отображаются красным цветом. Также можно выводить отладочные сообщения командой console.log().
